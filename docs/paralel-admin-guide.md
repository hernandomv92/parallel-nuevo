# Guia Operativa Del Admin

## Flujo recomendado

1. Crea o revisa categorias raiz: `Hombre`, `Mujer`, `Ninos`, `Deportes`.
2. Define atributos antes de cargar catalogo: `marca`, `genero`, `deporte`, `color`, `talla`, `coleccion`.
3. Crea colecciones para landings editoriales o marcas destacadas.
4. Carga productos con imagen principal, galeria, precio final, atributos y variantes.
5. Crea paginas CMS para soporte y legal.
6. Usa widgets cuando quieras que negocio edite bloques sin tocar el theme.

## Regla simple

- Categoria: navegacion y URL.
- Atributo: filtro y metadato operativo.
- Variante: cambio real de color o talla.
- Coleccion: campaña o curaduria.
- CMS: contenido informativo.

## Checklist por producto

- Categoria principal asignada.
- Marca definida como atributo.
- Genero definido como atributo o categoria raiz coherente.
- Deporte definido si aplica.
- Color y talla cargados de forma consistente.
- Imagen principal y galeria limpias.
- Precio en COP correcto.
- Descripcion util, no solo copiada sin limpiar.
